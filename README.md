# -Entrega-Projeto-Aula-Pensamento-Computacional-para-Sistemas-de-Larga-Escala-

Sistema Proposto
Nome do Sistema
  Plataforma Inteligente de Consultas Nutricionais

Descrição
  Uma aplicação web para conectar pacientes e nutricionistas, permitindo o acompanhamento nutricional de forma prática e escalável.

sistema integra:
  Cadastro e autenticação de usuários (pacientes e nutricionistas)
  Agendamento de consultas online
  Registro de planos alimentares personalizados
  Histórico de consultas e evolução do paciente
  Sistema de recomendações nutricionais automatizadas
  Pensamento Computacional Aplicado
  Decomposição

Metodologia de Desenvolvimento
Metodologia: Kanban (Fluxo Contínuo)

Em vez de trabalhar com sprints fixas, o projeto utiliza Kanban, focando em entrega contínua e melhoria progressiva.
  Organização do Fluxo
    Backlog → tarefas planejadas
    A Fazer (To Do) → tarefas prontas para iniciar
    Em Progresso (Doing) → desenvolvimento ativo
    Em Revisão (Review) → validação/testes
    Concluído (Done) → finalizado
  Princípios aplicados
    Limitação de tarefas em progresso (WIP)
    Entrega contínua de funcionalidades
    Feedback rápido
    Priorização dinâmica
  Ferramentas utilizadas
    GitHub Projects (Kanban Board)
    Issues (controle de tarefas)
    Git (versionamento de código)
